En este proyecto he utilizado un sistema de carpetas dependiendo de si son imágenes, hojas de estilo o javascript:


--Hay una carpeta llamada ASSETS en la cual están todas las imágenes utilizadas dentro del trabajo.

---Una carpeta llamada JS en el que está el archivo .js con el código de Javascript. 

--Una carpeta llamada SCSS en el que, dentro de ella, hay un archivo .scss (Los estilos han sido trabajados en SASS) y otra carpeta llamada RESET. En RESET hay diferentes hojas de estilo para keyframes, variables, mixins y una reset para cuestiones generales de estilo. 

--Un archivo de index.html en el que está el html principal.
